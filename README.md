# SUREK: The Architecture of Performance ⚡

**SUREK** es más que una plataforma de comercio electrónico de ropa deportiva de alto rendimiento; es un ecosistema híbrido diseñado para la evolución física y la independencia financiera a través de una economía descentralizada.

## 🚀 Sobre el Proyecto

Este repositorio contiene el código base del frontend para el ecosistema web de SUREK, construido con un enfoque en diseño de alta gama ("Premium Athletic" y estética "Cyberpunk"), interactividad avanzada y retención de usuarios.

La plataforma no solo vende productos físicos (Print-on-Demand) y digitales, sino que alberga el **Modelo de Monetización SUREK TV**: un sistema de "Turnos de Trabajo" donde los usuarios interactúan con contenido local de Puerto Rico para viralizarlo a nivel mundial, ganando ingresos pasivos y comisiones (Split 60/40) por ventas de ropa.

## ✨ Características Principales (Features)

*   **E-Commerce "White Label":** Catálogo interactivo de prendas físicas de alto rendimiento y productos digitales (The 90-Day Money Blueprint).
*   **Economía Descentralizada (SUREK TV):** Sección interactiva con una calculadora dinámica de ganancias pasivas basada en el compromiso (horas/semana) y el rol del usuario (Trabajador, Creador, Élite).
*   **Protocolo de 90 Días (Híbrido):** Un manual interactivo (acordeón de 9 páginas) que guía al usuario día por día en su transformación física (hipertrofia, nutrición) y financiera (cómo monetizar sus videos con los enlaces de afiliado de SUREK).
*   **Mentor IA Integrado:** Un chat interactivo diseñado para brindar asesoría instantánea sobre hipertrofia, recuperación, rutinas mañaneras y enfoque productivo.
*   **Diseño Dinámico:** Animaciones CSS avanzadas (Scroll reveals, contadores de métricas en tiempo real, microinteracciones en botones y tarjetas) para maximizar la retención.

## 🛠️ Tecnologías

*   **Estructura:** HTML5 Semántico
*   **Estilos:** Vanilla CSS3 (Animaciones personalizadas, CSS Grid, Flexbox, UI de Cristal/Glassmorphism).
*   **Lógica y Estado:** Vanilla JavaScript (ES6+).
*   **Modelo de Negocio Integrado:** Preparado para conexiones API con servicios de Print-on-Demand (ej. Printify) para dropshipping "White Label" automático.

## 🌍 Visión del Negocio
Transformar a los clientes en *creadores autosustentables*. SUREK provee la ropa, la logística y la red; el usuario aporta el esfuerzo físico y la distribución orgánica. El resultado es un motor de ventas impulsado por la comunidad.
